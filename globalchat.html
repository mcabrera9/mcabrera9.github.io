Global Chat - In Progress!

Redesign of Whatsapp

Summary

This project is an exploration in UI design and UX research practices as coursework for Computer-Human Interaction under supervision by Loyola University Chicago faculty member Dr. Nicholas Hayward. This project was inspired by my personal usage of WhatsApp across timezones, and my interest in finding solutions for the current issues that WhatsApp has.



Scope

The aim of this study is to practice wireframes, prototyping, and user experience research. The scope of this project is to focus on improving the current design, and incorporating additional functionality in an intentional way. My redesign will incorporate material design practices and considerations for accessibility.


Initial Research

To begin this project, I conducted an initial survey to inform my designs. I created a survey in Google Forms to gather data about how people use messaging applications to communicate, and how people use those same applications to send messages internationally.


Low-Fidelity Wireframes

The low-fidelity mockups were done by hand and list out my initial considerations for how I would like to lay out the design for the group chat settings page and the group chat interface.


Initial Prototype

This is the first set of prototypes I developed for this project. These screens were produced in Adobe XD. The goal with this prototype is to lay out all the features of the redesign. I did not incorporate colour because I wanted to first focus on the design before I fine-tune the intricacies of the design.


This is an animated gif of the first paper prototype I developed for the project. The purpose of this prototype and the animation is to demonstrate the planned flow of the application.
